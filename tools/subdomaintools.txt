# 1. Install Go tools (Subfinder, Amass, Puredns, Alterx, Github-subdomains, Hakrevdns)
go install -v github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
go install -v github.com/owasp-amass/amass/v4/...@master
go install -v github.com/d3mondev/puredns/v2@latest
go install -v github.com/projectdiscovery/alterx/cmd/alterx@latest
go install -v github.com/gwen001/github-subdomains@latest
go install -v github.com/hakluke/hakrevdns@latest

# 2. Install Waymore (Python tool)
pip install waymore
install dependency of every tools and dowload required files like worldlist for brutforce for finding subdomain.

instruction to run

​Passive: subfinder + amass + github-subdomains -> Save to list_1.txt
​Archives: waymore (on the main domain) -> Save found subs to list_2.txt
​Brute Force: puredns (using a big wordlist) -> Save to list_3.txt
​Reverse: hakrevdns (on the company IP range) -> Save to list_4.txt
​Permutation: Combine lists 1-4, remove duplicates, then run alterx on them -> Save to list_5.txt
​Final Validation: Combine ALL lists and run puredns resolve or dnsx to see which ones are actually alive.

save final file named activesubdomain.txt in output folder , all other files should be save in temp folder.
final file must not have duplicates,  it contains only live subdomain.
